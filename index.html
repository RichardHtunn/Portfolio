Infrastructure Drone Payload

ESP32 • OV2640 • Python • Systems Integration

The System Architecture

Instead of building a flight controller from scratch, I engineered a custom "vision payload" utilizing an ESP32 microcontroller and an OV2640 camera module. This payload acts as an independent subsystem mounted to the carrier drone.

  • Carrier Platform: Off-the-shelf commercial drone handling flight kinematics.
  • Vision Sensor: OV2640 Camera Module interfaced via FPC connector.
  • Electrical Isolation: The payload operates independently to prevent drawing excessive current from the primary flight controller.

Computer Vision Implementation

The ESP32 broadcasts a web server containing the live camera feed. On the ground station, a Python script utilizes OpenCV to process incoming video frames, applying image thresholding to highlight structural anomalies.

"Rhythm Hero" Arcade Game

FPGA (Basys3) • Raspberry Pi • VHDL • Digital Logic

Hardware-Software Synchronization

This project relies on a distributed hardware model to handle high-speed visual rendering and complex audio playback simultaneously.

  • Visual Engine (FPGA): A Basys3 board running custom VHDL handles the game state machine and generates 12-bit VGA timing signals.
  • Audio Management (Raspberry Pi): Receives 3.3V logic triggers via GPIO to manage multi-track audio playback with zero latency.
  • Input Debouncing: Physical arcade buttons are routed through hardware debouncing logic synthesized on the FPGA.
